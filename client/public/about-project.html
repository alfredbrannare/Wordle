About the Project

Introduction

This is a Wordle-inspired game built as a fullstack application using React and Node.js.

Before the game starts, the user can customize the experience by choosing:

  • The length of the word
  • Whether each letter must be unique or not

After successfully guessing the correct word, players can submit their result to a highscore list that stores data persistently in a database.

Purpose

The game was developed as part of my fullstack education. The main focus was to practice working with:

  • Building a frontend application using React
  • Creating a backend with Node.js and Express
  • Connecting and interacting with a database
  • Handling API communication between client and server
  • Implementing server-side rendering

Techniques Used

The following tools and technologies were used in the project:

  • React for the frontend and game UI
  • Node.js + Express for backend and API handling
  • MongoDB (via Atlas) for data persistence
  • EJS for server-side rendering of the highscore page
  • TailwindCSS and DaisyUI for styling and layout

Core Functionality

  • Randomly generated word based on user settings
  • Color-coded feedback for each letter guessed
  • Timer measuring duration from first guess to correct answer
  • Server-side rendered highscore list
  • Option to submit name and result after game completion

Summary

This project challenged me in many ways, especially with managing state and server-client interaction. However, I learned a lot throughout the process.

If I were to build this again, I would approach several parts differently, and I now feel more confident in developing fullstack applications from scratch.
